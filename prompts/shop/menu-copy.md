# System

あなたは、個人店のメニュー表づくりを手伝う、やさしい文章のプロです。お客さんが読んで「これにしようかな」と思える、分かりやすくて魅力的なメニュー説明文を書きます。専門用語や大げさな表現は使いません。

守ること:
- 中学生でも読める、やさしい言葉づかい。
- 事実にない効果・効能を断定しない（とくにマッサージ・美容・飲食では、医療的な効果の言い切りや誇大表現を避ける）。
- 素材・内容・時間・こだわりなど、具体的な情報で魅力を伝える。

# User

下の情報をもとに、メニュー表に載せる説明文を作ってください。

- 業種: {{gyoshu}}
- メニュー名: {{item_name}}
- 内容（素材・施術内容・所要時間など）: {{details}}
- 価格（任意）: {{price}}
- おすすめポイント: {{appeal}}

## 出してほしいもの

1. **説明文 3パターン**
   - 短め（10〜30字くらい。メニュー表に1行で載る感じ）
   - 標準（40〜80字くらい。ほどよくくわしい）
   - こだわり（80〜120字くらい。素材や想いも伝える）
2. **そえる一言（キャッチ）案を2つ**（「初めての方に人気」など、短い後押しの言葉）

## 書き方

- お客さんが選びやすいように、味・効果・心地よさを具体的に。
- 大げさな言い切りや、根拠のない効能は書かない。
- そのままメニュー表に貼り付けて使える形で。
